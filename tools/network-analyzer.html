What this tool does: Enter any domain to map its infrastructure network — SSL certificate neighbors, shared IP hosting, WHOIS clusters, and content relationships. Scores every domain on five independent dimensions regardless of country of origin. BBC, RT, CGTN, VOA: analyzed identically.

Graph-first methodology

The network graph is the primary output. Scores annotate the graph — they don't drive it. A domain with high connection density and moderate scores often reveals more than an isolated domain with a high score. Build the map, then read the dimensions.

Five scoring dimensions

Infrastructure Opacity — how hidden is hosting/WHOIS? Ownership Transparency — staff, funding, legal entity? Content Sourcing — original vs. republished vs. AI-laundered? Network Centrality — how connected in this graph? State Media Proximity — links to any state outlet, any country?

Nation-neutral coverage

The state media registry covers 10 countries symmetrically: CN, RU, US, GB, FR, DE, QA, IR, KP, VE. A high State Media Proximity score means "close to state media" — not "dangerous." The BBC scores high on this dimension because it is state-adjacent. Transparency dimensions distinguish it from covert operations.

How to use

Enter a seed domain and click Analyze →. The tool resolves its IP, ASN, SSL certificate SANs, WHOIS, and scrapes content signals. Neighboring domains auto-populate the queue. Run them to expand the graph. Click any node or table row to open the detail drawer. Export when ready.

Sources: Citizen Lab Report 174 (PAPERWALL) · Graphika Falsos Amigos (2025) · DFRLab/CheckFirst Pravda Network · VIGINUM Portal Kombat. This tool produces signals, not verdicts.

Investigate: analyze one domain and manually expand. Expand: analyze seed then automatically run first 10 neighbors.
