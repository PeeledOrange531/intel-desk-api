WHOIS & DNS Lookup

What This Tool Does

Every domain name on the internet has a registration record and a DNS configuration — and both contain intelligence that investigators, security researchers, and journalists rely on daily. This free WHOIS and DNS lookup tool performs a comprehensive domain investigation in one query. The WHOIS lookup reveals the registrar, registration and expiry dates, nameservers, registrant contact details (where not redacted), and domain status codes. The DNS lookup — powered by Cloudflare's DNS-over-HTTPS resolver — returns all record types including A records (IP addresses), MX records (mail servers), TXT records (SPF, DKIM, DMARC, verification tokens), NS records (nameservers), CNAME, and AAAA (IPv6). Used by security researchers investigating phishing infrastructure, journalists tracing domain ownership, fraud analysts verifying counterparty legitimacy, IT professionals diagnosing email delivery issues, and OSINT investigators mapping digital infrastructure.
