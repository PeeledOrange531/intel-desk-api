What This Tool Does

No single detector is reliable enough to use alone. This tool runs five independent signals simultaneously and combines them into a weighted confidence score. Hive AI uses a dedicated neural network trained specifically on AI-generated images and deepfake face manipulation. HuggingFace inference runs an independent model trained on Stable Diffusion, DALL-E, and Midjourney outputs. Error Level Analysis (ELA) is a classical forensics technique that reveals JPEG recompression artifacts — manipulated or composited regions show different error levels than authentic content. Metadata forensics scans the file for AI generator signatures embedded by tools like Stable Diffusion, Midjourney, and DALL-E. C2PA / Content Credentials checks for embedded provenance metadata from camera manufacturers and content platforms. No single signal is definitive — the tool shows you all signals with their individual confidence levels so you can make an informed judgment.
