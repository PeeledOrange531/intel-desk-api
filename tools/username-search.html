What This Tool Does

A username is a digital fingerprint — most people reuse the same handle across platforms, creating a traceable thread through their entire online presence. This tool runs two industry-standard open-source intelligence engines: Sherlock checks for a username across 300+ social networks and platforms including Twitter, Instagram, Reddit, GitHub, TikTok, LinkedIn, and hundreds more. Maigret goes deeper — scanning 3000+ sites and extracting profile metadata, linked accounts, and personal information where publicly available. Results stream live into the terminal as each match is found, with direct clickable links to every discovered profile. No API key, no login, and no data is stored. Used by journalists investigating sources, fraud analysts verifying identities, HR professionals screening candidates, security researchers tracking threat actors, and investigators building subject profiles.
