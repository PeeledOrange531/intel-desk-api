What This Tool Does (and What It Cannot Do)

Free username permutation generator and cross-platform availability checker for OSINT investigators, brand protection teams, and security researchers. Paste any base username and the tool generates dozens of categorized variations — numeric suffixes (jane2024, jane99), leetspeak substitutions (j4ne, jan3), separator inserts and strips (jane_doe, janedoe), prefix/suffix words (real_jane, jane_official, jane_hq), doubled-letter typos (jaane, jannie), truncations (jan, janed), phonetic swaps, case variants, and reversals. Each permutation is scored 0–100 by a heuristic that ranks how likely the variation is to be a real alt-handle (year suffixes and "real/official/hq" suffixes rank highest). The Quick Check mode probes ~10 high-reliability platforms; Deep Check hits 25+ platforms; Brand Protection mode focuses on impersonation-risk variants and runs a "squat signal" check on the highest-value platforms (low Reddit karma, empty GitHub profiles). The Reverse Pattern mode takes a list of known handles and extracts the common base + naming convention so you can predict the next likely handle. Honest framing throughout: we label results as Taken / Not Found / Unverifiable — never "Available" — because a 404 doesn't guarantee a username is registrable (it may be reserved, deactivated, banned, or simply protected). Platforms that require JavaScript or login (Instagram, TikTok, LinkedIn, X, Threads, Facebook) are flagged as low reliability with explicit notes — we attempt the probe but the result is best-effort. All data sources are public HTTP probes; no API keys, no signup, no logging of inputs. Used by journalists tracking sock-puppet networks, brand and IP protection teams, investigators identifying typo-squat domains and handles, and analysts mapping coordinated inauthentic behavior.
