What This Tool Does

A domain reveals far more than its current website. This tool aggregates intelligence from four independent public sources simultaneously. Certificate Transparency logs (via crt.sh) expose every subdomain that has ever been issued an SSL certificate — revealing forgotten staging servers, internal tools, and shadow infrastructure. The Wayback Machine CDX API returns every URL the Internet Archive has ever crawled for a domain, revealing deleted pages, old URL structure, and content history going back decades. Common Crawl provides real backlink data — pages across the web that link to the target domain. DNS records reveal mail servers, SPF/DMARC configuration, nameservers, and IP addresses. Used by OSINT investigators profiling organisations, penetration testers mapping attack surface, journalists researching corporate infrastructure, and security researchers tracking domain history.
