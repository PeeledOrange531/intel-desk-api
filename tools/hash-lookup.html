What This Tool Does

Free malware hash intelligence with deep file analysis. Paste an MD5, SHA1, or SHA256 hash and the tool queries three abuse.ch threat databases in parallel — MalwareBazaar (full malware metadata, family, signature, tags), ThreatFox (IOC database with confidence scoring), and URLhaus (distribution URLs that have served the file). Or drop a file directly — it is hashed in your browser using Web Crypto API (the file never leaves your machine for hashing) then a small server-side analysis pipeline runs: file-type detection by magic bytes (catches `.pdf` files that are actually executables), Shannon entropy (high values indicate packed or encrypted payloads), YARA scanning against the community Yara-Rules ruleset (thousands of rules covering known malware families), VBA macro detection for Office documents, JavaScript and auto-action detection for PDFs, suspicious string scanning for known malicious API patterns, and IOC extraction (URLs, IPs, email addresses embedded in the file). Drop a .eml file and every attachment is extracted and analyzed individually. Drop a ZIP/RAR/7Z archive and every member file is unpacked and scanned. Bulk hash mode processes up to 100 hashes in parallel with CSV export. Used by SOC analysts, journalists investigating malware campaigns, incident responders, and security researchers.
