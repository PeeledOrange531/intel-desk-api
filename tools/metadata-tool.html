What This Tool Does

Free metadata extractor and scrubber for Office documents, PDFs, images, audio, and video. The Extract mode reveals every hidden field embedded in your file — author identity, last modified by, company name, GPS coordinates, device serial numbers, file paths leaked from the original computer, creator software fingerprints, full EXIF/IPTC/XMP data on images, ID3 tags on audio, tracked changes and reviewer comments in Word docs, hidden slides in PowerPoint, hidden sheets/columns in Excel, PDF revisions that may contain pre-redaction text, and embedded files hidden inside other documents. Each file gets a 0-100 leakiness score that tells you exactly how much identifying information you'd expose by sharing it. The Scrub mode strips all metadata using ExifTool then returns a downloadable cleaned version. Bulk modes handle up to 25 files at once. Compare mode diffs metadata between two files — spot when someone modified a leaked document, or verify two files came from the same source. Revision history reconstruction shows you previous versions of PDFs (including text that was supposedly redacted) and full Office edit history. Used by journalists protecting sources, activists sharing documents safely, investigators verifying leaked files, and anyone publishing files who doesn't want their identity, location, or organization leaked along with the content.
