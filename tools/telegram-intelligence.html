Telegram OSINT — Investigation Techniques
🔍
Finding a Channel's Numeric ID
Forward any message from a channel to @userinfobot or @getidsbot to retrieve the permanent numeric ID. Unlike usernames, this ID never changes and is useful for tracking entities that rename themselves. Channel IDs typically start with -100.
🕰
Account Creation Date from ID
Telegram user IDs are sequential — lower numbers indicate older accounts. Use @tgdb_bot or @creationdatebot to estimate when an account was created from its numeric ID. Pre-2013 IDs (below ~10M) are very early adopter accounts.
🌐
Google Dorking for Telegram Content
Use site:t.me "keyword" in Google to find public Telegram content indexed by search engines. Combine with inurl:t.me for channel links. Try site:t.me/s/ "phrase" to search within channel message previews that Telegram exposes publicly.
📸
Profile Photo Intelligence
Profile photos can be downloaded from any public account. Run them through Reverse Image Search (another Intel Desk tool) — particularly Yandex, which excels at finding Russian-language sources. Profile photos are often reused across platforms and can link identities together.
🔗
Forward Chain Analysis
When a channel forwards messages, the original source is preserved and linked. Building a forward map — which channels forward to/from which others — reveals networks and relationships. Tools like @forwardmapbot automate this. The Telepathy framework on GitHub does this at scale.
📍
Geolocation via Nearby Feature
Telegram's "People Nearby" feature exposes users and groups within a radius. This feature has been partially restricted since 2024, but archived data from tools like Geogramint may still be useful for historical investigations. Always check the date of any location-based data.
⚠
Account Takeover & Impersonation Indicators
Sudden change in posting language, tone shift, or content pivot may indicate account takeover or sale. Check TGStat for historical subscriber counts and posting frequency. A sudden spike in subscribers combined with a content change is a strong impersonation indicator.
