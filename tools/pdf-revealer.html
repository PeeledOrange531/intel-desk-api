What This Tool Does

Free PDF forensics tool that reveals content invisible when the PDF is opened normally — but still embedded in the file. The Reveal All mode runs eight independent extraction layers on a single uploaded PDF: failed redaction detection uses geometric coordinate matching to find text positioned underneath filled black or white rectangles (the classic "Acrobat black box over text" failure that has exposed names in court filings, NSA documents, and corporate depositions); Optional Content Group (OCG) layer extraction lists every PDF layer including ones configured as default-off, which CAD drawings and technical documents commonly hide from casual viewers; off-page text detection finds text positioned outside the visible page boundary (clipped from view but extractable); JavaScript extraction dumps every embedded JS block; annotation extraction pulls all comments, sticky notes, highlights with author + timestamp; form field analysis shows both default and current values for every field; incremental revision walking reconstructs each previous version of the PDF (PDFs use append-only updates, so "deleted" content often remains in earlier sections); visible-vs-actual page rendering shows the PDF as a casual viewer sees it side by side with how it renders when all hidden content is surfaced. Each PDF gets a 0-100 hidden content severity score. Used by journalists analyzing leaked documents, lawyers reviewing produced files, activists verifying redactions, FOIA requesters checking what was withheld, and OSINT investigators who know that the most interesting parts of a document are often the parts someone tried to hide.
