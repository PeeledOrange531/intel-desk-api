What This Tool Does (and What It Cannot Do)

Free X (Twitter) account forensics tool that operates entirely on public archive sources — no X API key required. The tool checks the Wayback Machine CDX API for every archived snapshot of a profile (often back to 2008), queries memory.lol for the account's full handle rename history (catches identity-laundering — when accounts change handles to deflect attention from past activity), and runs a live HTTP probe against both x.com and twitter.com to detect whether an account is currently alive, suspended, deleted, or has never existed. The tool provides a defensible lower-bound date for account age based on archived evidence — not the actual creation date (which is only available via paid X API), but a date by which the account is provably known to have existed. Tweet recovery mode takes any tweet URL or status ID and finds Wayback snapshots that captured it before deletion. Bulk mode processes up to 50 handles in parallel for tracking lists of accounts, watchlists, or research targets. Fingerprint comparison mode analyzes two handles for sock-puppet patterns including shared rename history, mutually exclusive activity periods, and overlapping archive timelines. Suspicious-pattern detection automatically flags accounts with no archive history (suspicious if alive — may indicate scrubbing or very new accounts), frequent renames (identity laundering), and dormancy followed by reactivation. Used by journalists tracking sources and disinformation networks, OSINT researchers verifying account claims, investigators recovering deleted public statements, and analysts monitoring extremist or coordinated inauthentic behavior.
