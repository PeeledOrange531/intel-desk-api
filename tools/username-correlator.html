What This Tool Does (and What It Cannot Do)

Free cross-platform username correlation tool for OSINT investigators tracking sock-puppet networks, identity-laundering campaigns, and coordinated inauthentic behavior. Computes a 0–100 same-person likelihood score from four weighted lexical signals: Jaro-Winkler similarity (40% — heavy on shared prefixes, robust for short strings), n-gram overlap (25% — bigram + trigram Jaccard average), shared core (20% — what remains after stripping leading/trailing digits and separators), and substitution detection (15% — leetspeak normalization plus separator-stripped equality). The scoring is also computed against the leet-normalized forms (so j0hnsmith and johnsmith match cleanly) and the alphanumeric-only forms (so john_smith and johnsmith match). Pair Compare mode shows a full signal breakdown with weighted contribution bars so you can see exactly why two handles scored as they did. Cluster mode takes up to 20 handles, builds the full pairwise matrix, and groups them via union-find at an adjustable threshold — visualized as a D3 force-directed graph with weighted edges. Sock-Puppet mode adds Wayback Machine CDX presence probes on Twitter/X/GitHub/Reddit/Instagram (mutually exclusive activity windows are a strong sock-puppet signal — one account goes dormant exactly when another lights up), plus memory.lol queries to detect handles that share historical X identities (identity-laundering signal). Pattern Extract mode finds the common base, naming convention (year suffixes, separators, casing), and predicts next-likely handles. Honest framing throughout: the bands are labeled almost certainly same / likely same / possibly related / weak signal / different — never "match" or "confirmed." Lexical similarity tells you the same brain selected these handles, not that the same person operates them. Short handles (≤3 chars) are capped at 60 points because shared 3-letter prefixes are statistically meaningless. All data sources are public — no API keys, no signup. Used by journalists tracking disinformation networks, OSINT analysts mapping coordinated inauthentic behavior, brand protection teams identifying impersonator clusters, and investigators correlating accounts across forums and social platforms.
