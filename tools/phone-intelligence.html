What This Tool Does

This tool provides two layers of phone number intelligence. The first layer uses Google's libphonenumber — the same library used by Android and WhatsApp — to instantly extract carrier, line type (mobile, landline, VoIP, toll-free), country and region, timezone, and all standard formatting variants (E.164, national, international). The second layer runs Ignorant, an open-source tool that checks whether a phone number is registered on platforms including Instagram and Snapchat using their account-recovery flows — without alerting the number's owner and without requiring any login. Results from the platform scan stream live into the terminal. This combination gives investigators a complete initial profile of any phone number: what it is technically, where it's from, and where it has a digital presence. Used by fraud investigators, journalists, due diligence analysts, and OSINT researchers.
