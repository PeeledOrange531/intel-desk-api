Influence Network Tracer

Fingerprint state media affiliate networks via shared infrastructure — ASN clustering, SSL certificate overlap, analytics IDs, and content signals.

What this tool does: Enter any suspicious domain to trace its infrastructure back to known state media operators — or start from a known actor like CGTN to map its affiliate network. Every signal is source-cited from published research. — methodology from Citizen Lab, Graphika, DFRLab.

Why this tool matters

Networks like PAPERWALL (Citizen Lab, 2024) and Falsos Amigos (Graphika, 2025) operate hundreds of fake local news websites that evade detection by appearing independent. They are connected by invisible infrastructure threads — shared hosting providers, identical analytics IDs, SSL certificates that list every domain in the network, and outbound links to state media hubs. This tool automates the forensic methodology used by professional researchers to surface those connections.

How to use it

Investigate a domain (bottom-up): Paste a suspicious URL. The tool resolves its hosting ASN, harvests its SSL certificate's Subject Alternative Names, checks its WHOIS registrar, scrapes its HTML for analytics IDs and state media backlinks, and scores the result against a cited database of known signals. A score above 50 means multiple independent signals align — not a single heuristic.

Map from a state actor (top-down): Select a known state media outlet. The tool expands outward through SSL SANs and shared IP infrastructure to surface domains that may be part of the same network — potentially including sites not yet reported publicly.
